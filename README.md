
# 📊 Sales Performance Dashboard using Microsoft Excel & VBA

## 📌 Project Overview

This project is an interactive Sales Performance Dashboard built using Microsoft Excel, Pivot Tables, Pivot Charts, Slicers, and VBA.

The dashboard provides insights into:

- Monthly Sales Trend
- Product-wise Sales
- Salesperson Performance
- Regional Profit Analysis
- KPI Cards
- Dynamic Dashboard Filtering
